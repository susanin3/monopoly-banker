# Monopoly Banker

Monopoly Banker is a small web application for tracking and managing players finances during a game of Monopoly. It helps record balances, transactions, and bank operations so you can focus on gameplay instead of manual bookkeeping.

## Features

- Keep accurate player balances (payments, rent, property purchases, taxes, fines).
- Record and undo transactions to correct mistakes (in the future).
- Manage bank cash flow and property-related transfers (in the future).
- Support multiple players and quick game setup.

See the Setup section below to install dependencies and run the development server or build the project.

#### ***You can also use it online: https://mpb.qwwa.ru.***

Works entirely locally. No registration required.

## Setup

Install the dependencies:

```bash
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
pnpm dev
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```
